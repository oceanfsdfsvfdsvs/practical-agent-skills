# OpenClaw Notes

Install by copying the `workslop-review/` directory into the OpenClaw workspace skills directory.

Suggested check when supported by your CLI:

```bash
openclaw skills check workslop-review
```

This is a prompt/rubric skill with no local script. Validate by asking the agent to apply `workslop-review/templates/review-rubric.md` to `workslop-review/examples/bad-status-update.md`.
