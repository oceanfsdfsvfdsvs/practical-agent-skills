# Medical Bill Dispute Packet Checklist

## Case Snapshot

- Patient or authorized helper:
- Provider/facility:
- Date(s) of service:
- Bill/account number:
- Insurance claim/EOB number:
- Current balance:
- Review date:

## Evidence To Gather

- Provider bill or statement.
- Itemized bill or superbill with CPT/HCPCS and service dates.
- Insurance EOB for each claim.
- Insurance card copy if needed by provider.
- Prior authorization, referral, estimate, good-faith estimate, or in-network confirmation if relevant.
- Call log with dates, phone numbers, representatives, and reference numbers.
- Written provider or insurer messages.

## Issue Classification

| Issue | Evidence | Owner to contact | Requested action | Follow-up date |
|---|---|---|---|---|
| Bill exceeds EOB patient responsibility |  | Provider billing | Reconcile account to EOB |  |
| No EOB or unprocessed insurance |  | Provider and insurer | Submit/reprocess claim |  |
| Missing itemized bill |  | Provider billing | Send itemized statement |  |
| EOB denial or noncovered line |  | Insurer | Send appeal procedure and evidence requirements |  |
| Surprise-billing clue |  | Provider, insurer, No Surprises Help Desk/state DOI | Review compliance path |  |
| Unaffordable balance |  | Provider financial assistance | Screen charity care, self-pay, or payment plan |  |

## Written Request Draft

I am reviewing this medical bill before payment. Please provide written clarification for the disputed line items listed above, including any corrected claim, reprocessed EOB, itemized bill, or account adjustment. Please pause collection activity on the disputed balance while this review is pending.

## Guardrails

- This packet is administrative support, not legal or medical advice.
- Do not include full SSNs, full member IDs, payment card data, portal passwords, or unrelated clinical records.
- Keep final payment, appeal, complaint, and legal decisions with the patient or authorized representative.
